# Fashion-E-commerce-Sales-Customer-Insights-Dashboard
This project showcases an end-to-end data analysis workflow where raw e-commerce data is transformed into actionable insights using interactive dashboards.

# 🔍 Key Highlights
Cleaned and structured raw dataset for analysis


Built an interactive Excel dashboard for business insights


Identified key performance indicators (KPIs) such as:

  -> Total Sales & Revenue Trends
  
  -> Top Performing Products
  
  -> Customer Purchase Behavior
  
  -> Regional Sales Distribution


Enabled dynamic filtering for better data exploration
